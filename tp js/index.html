Mini Calculatrice JavaScript












Grâce à ce TP, j’ai appris à manipuler le DOM avec JavaScript, à créer des fonctions pour gérer les événements comme les clics sur les boutons, et à séparer le code HTML, CSS et JS. J’ai aussi compris comment contrôler les entrées utilisateurs, effectuer des calculs simples, et remettre un formulaire à zéro. Ce travail m’a permis de mieux comprendre le lien entre la logique et l’affichage.
